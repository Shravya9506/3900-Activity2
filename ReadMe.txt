This website provides an insight of the ISQA 3900 Web Application development course catalog offered by University of Nebraska Omaha.

The index page of the web application describes briefly the topics being covered by the course and also provides the end users with an option to know more details about the courses such as course objectives, assignments and text book that provide good knowledge about the course, by clicking on the links provided on the index page.

The assignments.html web page provides information about the various assignments that will be conducted as part of the course work and the grading methodology for the course.

The textbooks.html page provides information about the books which are required and recommended for the course.
